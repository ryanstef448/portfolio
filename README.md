# Personal Data Science Portfolio

Welcome to my personal portfolio! This repository contains the source code for my portfolio website, showcasing my projects and skills in Data Science, Machine Learning, and Analytics.

## Sections
- **Home**: A brief introduction and welcome.
- **About Me**: My background, education (B.S. in Applied Data Sciences from Penn State University), and passion for data.
- **Projects**: A curated list of projects with links to their respective repositories/source code.
- **Skills**: A summary of my technical toolkit, including Python, SQL, and NLP.
- **Contact**: How to reach me for collaborations or inquiries.

## Featured Projects
1. **Disaster Tweet Classification**: Using NLP to identify emergency situations from social media.
2. **Predicting Movie Ratings**: Applying PCA and XGBoost to predict audience ratings.
3. **Visual Analytics**: Transforming complex datasets into intuitive visualizations.
4. **Exploratory Data Analysis**: Uncovering patterns and insights in diverse datasets.
