Predicting Movie Ratings

This project explores how machine learning can be used to predict movie ratings based on various features like budget, revenue, and popularity. By applying Principal Component Analysis (PCA), the study identifies key factors that influence ratings and simplifies the dataset for better model performance. The machine learning model is tested with and without PCA to assess its impact on prediction accuracy, providing insights into how dimensionality reduction can improve predictive modeling in the film industry.
